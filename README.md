# cryptoIQ

Full-stack cryptocurrency prediction platform that combines traditional machine learning (lstm), innovative pattern recognition (financial DNA) and AI agents for market sentiment analysis.

Technology Stack
Backend
    •	TypeScript/Node.js - API Gateway, business logic, data orchestration
    •	Python + FastAPI - ML model serving and inference
    •	PostgreSQL + TimescaleDB - Time-series data storage
    •	Docker + Docker Compose - Containerization

Frontend
    •	React + Next.js - UI framework
    •	NextUI + Tailwind CSS - Styling and components
    •	Recharts/Chart.js - Data visualization

Machine Learning & AI
    •	PyTorch - Deep learning framework
    •	LSTM Networks - Time-series forecasting
    •	Financial DNA - Pattern recognition via sequence alignment
    •	BioPython - Sequence alignment algorithms
    •	Mistral AI API - Market sentiment analysis (conditional)
